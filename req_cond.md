# System Requirements: Vehicle HVAC (Air Conditioner)

## 1. System Overview
Система клімат-контролю для підтримки заданої температури в салоні.

## 2. Operational States (Робочі стани)
- **Standby**: Система увімкнена, але вентилятор і компресор не працюють.
- **Cooling**: Температура в салоні вища за цільову (`CabinTemp > TargetTemp`). Увімкнено компресор.
- **Heating**: Температура в салоні нижча за цільову (`CabinTemp < TargetTemp`). Увімкнено обігрівач/відкрито кран радіатора.

## 3. Safety Triggers (Тригери безпеки)
- **PressureFault**: Критично високий або низький тиск фреону в системі.
- **CompressorOverload**: Перевантаження або заклинювання компресора.

## 4. Specific Constraints (Специфічні обмеження)
- При `PressureFault` система повинна негайно знеструмити електромагнітну муфту компресора, щоб уникнути пошкодження магістралі.

## 5. Environmental Constraints (Умови експлуатації)
- Забороняється активація компресора (стан `Cooling`), якщо температура зовнішнього повітря (`OutsideTemp`) становить нижче 2°C, для запобігання обмерзанню випарника.
